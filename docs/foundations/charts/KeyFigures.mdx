# Key figures (text)

A big number with an icon and a one-line label. The most fundamental — and most under-used — OCHA dataviz format. Pair with a source line.

## When to use

- A single number is the headline (people in need, funding gap, children reached).
- The KPI row at the top of a dashboard, brief, or infographic.
- 1–5 figures in a row. More than 5 — break to a second row.
- Avoid decimal precision unless it matters (`29.8M`, not `29,762,793`).
- Avoid when the comparison is across categories — use a bar chart.

## Apply it in your tool

### Power BI
- Visual: **Card**. Number format: `k/M/B` (Modelling → Format).
- Font: Roboto Bold, 22–30pt. Color `#000000`.
- Add a **text box** below with the label (Roboto Condensed 11pt, `#000000`) and the source (Roboto 10pt, `#737373`).
- Pair with an OCHA icon via **Image** visual.

### Datawrapper
- Use a **table** with a single row and apply a custom heading; or embed a static infographic from the Dataviz Tool.

### Chart.js / D3
Not applicable — this is pure HTML. Use `<div style="font: 700 30px/1 Roboto; color: #000">29.8M</div>` above a `<div style="font: 700 12px/1.2 'Roboto Condensed'">People reached</div>`.

### Illustrator
Use the [Humanitarian Dataviz Tool](https://brand.unocha.org) — the Key Figures renderer auto-flows an icon grid.
