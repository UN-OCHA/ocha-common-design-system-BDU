# Map

Choropleth (admin regions shaded) or bubble (magnitudes at centroids). Uses the OCHA cartography style so every map across OCHA products reads the same way.

## When to use

- **Choropleth** — when a value is attached to an administrative region (PiN by admin 1, severity by district).
- **Bubble** — when you're showing magnitude without filling the whole region.
- Avoid when the reader needs precise ranking — pair with a small horizontal bar.
- Always include the UN disclaimer on disputed territories.

## Style values

| Element | Value | Token |
| --- | --- | --- |
| Ocean | `#E1E8F6` | — |
| Featured country fill | `#FFFFFF` | — |
| Surrounding countries | `#E6E7E8` | *(tool uses this; note DataViz.mdx mentions `#E6E6E6`)* |
| No-data (neutral) fill | `#E6E6E6` | `--cd-grey--400` |
| Admin 0 (international) | `#737373`, stroke-width `sw(1.3)` solid | `--cd-grey--700` |
| Admin 1 | `#C7C8CA`, stroke-width `sw(0.8)` dashed | — |
| Coastlines | `#64BEEB`, stroke-width `sw(0.5)` | — |
| Choropleth ramp | `#E3EDF6`, `#C5DFEF`, `#64BDEA`, `#009EDB`, `#0074B7`, `#004987`, `#002E6E` | OCHA ramp |
| Bubble color | `#009EDB`, fill-opacity `0.7`, white stroke 0.5pt | `--brand-primary` |
| Bubble max radius | `20px` default | `bubbleMaxRadius` |
| Disclaimer font | Roboto italic 7px, `#A7A9AC` | — |
| Capital marker | 4px circle, `--brand-primary` | — |

Stroke widths scale via `sw(ref) = round(ref × mapW/842 × 4) / 4`. Values from `chart-map.js`.

## Apply it in your tool

### Power BI
- Visual: **Shape Map** or **ArcGIS Maps for Power BI**.
- Ocean background: `#E1E8F6`. Admin 0 border: `#737373`, 1.3pt solid.
- Choropleth ramp: import the OCHA Blue ramp above (7 steps).
- Always add a text box with the UN disclaimer.

### Datawrapper
- Chart type: **Choropleth map** or **Symbol map**.
- Upload OCHA CODs boundaries. Apply OCHA theme (contact BDU). Legend: sequential blue, 5–7 bins.

### Chart.js / D3
```js
// D3 with topojson
svg.append('rect').attr('width', w).attr('height', h).attr('fill', '#E1E8F6');
svg.selectAll('.adm1').data(adm1).join('path').attr('d', path)
  .attr('fill', d => scale(d.value)).attr('stroke', '#C7C8CA').attr('stroke-width', 0.8)
  .attr('stroke-dasharray', '2.4 2.4');
svg.selectAll('.adm0').data(adm0).join('path').attr('d', path)
  .attr('fill', 'none').attr('stroke', '#737373').attr('stroke-width', 1.3);
```

### Illustrator
Use the [Humanitarian Dataviz Tool](https://brand.unocha.org) — the Map panel produces OCHA-compliant maps from CODs boundaries with the disclaimer baked in.
