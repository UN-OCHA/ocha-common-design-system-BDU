# Donut

Binary data only: funded vs. unfunded, reached vs. not reached, in need vs. targeted. The hole holds the headline number.

## When to use

- Exactly **two** categories — funded/gap, reached/not-reached.
- The headline number belongs in the hole.
- Avoid 3+ slices on a donut — use a pie (rarely) or a horizontal bar.
- Avoid when the comparison is between multiple donuts side-by-side.

## Apply it in your tool

### Power BI
- Visual: **Donut chart**. **Inner radius** `60%`.
- Two categories only. Series colors `#009EDB` + `#C5DFEF`. Outline `#FFFFFF` width `2`.
- Use a **card** visual overlaid in the centre for the headline number; Roboto 16pt bold.

### Datawrapper
- Chart type: **Donut**. Inner radius 60%.
- Add centre annotation via *Annotate → Text* at the chart midpoint.

### Chart.js / D3
```js
new Chart(ctx, {
  type: 'doughnut',
  data: { labels: ['Funded', 'Gap'], datasets: [{ data: [38, 62], backgroundColor: ['#009EDB', '#C5DFEF'], borderColor: '#FFFFFF', borderWidth: 2 }] },
  options: { cutout: '60%', plugins: { legend: { position: 'bottom' } } }
});
```

Use the [Humanitarian Dataviz Tool](https://brand.unocha.org) — it applies all these values automatically.
