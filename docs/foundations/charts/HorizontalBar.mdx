# Horizontal bar

Compares values across categories when labels are long (countries, sectors, organizations). The most versatile OCHA chart.

## When to use

- Categorical comparison with 4–15 items and long labels.
- Ranking (largest to smallest) is the story.
- Values share a common unit (people, USD, %).
- Avoid when the X axis is time — use a line or vertical bar instead.
- Avoid when you need to show part-to-whole with more than two segments — use stacked bar.

## Style values

| Element | Value | Token |
| --- | --- | --- |
| Primary bar fill | `#009EDB` | `--brand-primary` |
| Bar corner radius | `0` (sharp) | — |
| Bar thickness (md 350–550px) | `24px` | — |
| Bar gap | `10px` (md) | — |
| Margin left / right (md) | `50px` / `20px` | — |
| Title font | Roboto 14px, bold, `#1A1A1A` | `--cd-font--roboto` |
| Label font | Roboto Condensed 11px, `#555555` | `--cd-font--roboto-condensed` |
| Value font | Roboto Condensed 10px, `#333333` | `--cd-font--roboto-condensed` |
| Footer / source | Roboto 10px, `#999999` in-tool; use `#737373` on white for AA contrast | `--cd-grey--700` |
| Max label chars (md) | `20` (truncate) | — |
| Value label offset | `6px` outside bar | — |

Values from `chart-hbar.js` + `chart-registry.js` breakpoint `md`.

## Apply it in your tool

### Power BI
- **Format → Bars → Fill:** `#009EDB`. **Corner radius:** `0`.
- **Data labels:** On, Position **Outside end**, Font **Roboto Condensed 10pt**, Color `#333333`.
- **Sort axis:** descending by value.
- **X axis / gridlines:** Off. **Y axis line:** Off.

### Datawrapper
- Theme: **OCHA** (contact BDU for access).
- If building manually: *Appearance → Bars → Color* `#009EDB`; *Labels* → outside; *Axis* → hide gridlines and baseline; *Sort* by value.

### Chart.js / D3
```js
new Chart(ctx, {
  type: 'bar',
  data: { labels, datasets: [{ data, backgroundColor: '#009EDB', borderRadius: 0 }] },
  options: {
    indexAxis: 'y',
    plugins: { legend: { display: false } },
    scales: {
      x: { beginAtZero: true, grid: { display: false }, ticks: { font: { family: 'Roboto Condensed', size: 10 }, color: '#333333' } },
      y: { grid: { display: false }, ticks: { font: { family: 'Roboto Condensed', size: 11 }, color: '#555555' } }
    }
  }
});
```
