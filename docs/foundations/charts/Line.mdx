# Line

Shows a trend across time. The default choice for anything with a continuous X axis.

## When to use

- Time series with ≥ 4 points, evenly spaced.
- One trend (single-line), up to 4 trends (multi-line), or many trends shown as small multiples.
- Avoid when increments are too coarse (2–3 years) — a bar chart reads clearer.
- Zero baseline is **not mandatory** on a line chart — scale to show the trend.

## Variants

### Multi-line (up to 4)

Use the OCHA ramp in order: `#009EDB`, `#0074B7`, `#64BDEA`, `#004987`. Label each line at its endpoint — never a legend box in the corner when endpoints have room.

### Small multiples

When you have 5+ series, split into a grid. Same scale and X range across every panel. Highlight one series in `#009EDB`; grey the rest to `#E6E6E6` as reference.

## Apply it in your tool

### Power BI
- **Format → Lines → Stroke width:** `2`. **Color** `#009EDB`.
- **Markers:** On, Shape **Circle**, Size `4`, Fill `#FFFFFF`, Border color matches line, Border width `1.6`.
- **Data labels:** On at first & last point; others off.
- **Y axis:** allow non-zero baseline to reveal the trend.

### Datawrapper
- Theme: **OCHA**.
- Line chart: *Refine → Line thickness* `2`, *Symbols* at endpoints, *Label lines directly*.

### Chart.js / D3
```js
new Chart(ctx, {
  type: 'line',
  data: { labels, datasets: [{ data, borderColor: '#009EDB', borderWidth: 2, pointRadius: 4, pointBackgroundColor: '#FFFFFF', pointBorderWidth: 1.6, tension: 0 }] },
  options: {
    plugins: { legend: { display: false } },
    scales: { x: { grid: { display: false } }, y: { grid: { color: '#E6E6E6' }, border: { display: false } } }
  }
});
```
